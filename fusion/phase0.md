# Phase 0 — Fusion 360 基礎操作

> **期間目安**: 1週間（試用版残り8日：2026-05-31〜2026-06-07）
> **ゴール**: Fusion 360 を起動してスケッチ→3Dボディ→保存までを1人で迷わずできる
> **作品**: 簡単なペン立て（または好きな単純形状）を1個完成させる

---

## ★ 学ぶ前に — Phase 0 の位置づけ ★

Phase 0 は **操作の基礎体力作り**。
ここで指が覚えるべきは：

| 5大基本操作 | 何ができれば合格 |
|---|---|
| スケッチ | 線・円・矩形を描いて寸法を入れる |
| 押出 | 平面のスケッチを厚みのある3Dにする |
| 回転 | 半分の断面から回転体（コップ等）を作る |
| 視点操作 | マウスでくるくる回す・拡大縮小 |
| 保存・履歴 | クラウドに保存・前の状態に戻す |

**全部できれば Phase 1（板金・パラメトリック）に進める。**

## ★ つまずきやすいポイント（先回り）★

| 症状 | 原因 | 対処 |
|---|---|---|
| スケッチ線が黒くならない | 寸法・拘束が足りない（未確定線は青） | 寸法（D）と拘束を足して **黒線＝完全拘束** にする |
| 押出ができない | スケッチが閉じていない | 線の端と端が繋がってるか確認・隙間を埋める |
| ViewCube が変な向き | 視点が回りすぎた | ViewCube の「家マーク」で初期視点に戻る |
| ファイルが見つからない | ローカルじゃなくクラウド保存 | 画面左上 **データパネル** から探す |
| 操作が分からなくなった | UIに飲まれた | Ctrl+Z で1個ずつ戻す・タイムラインから戻る |
